# Navigating Shopper's Retail

This project is a retail analytics dashboard designed to provide insights into customer behavior, sales trends, and operational efficiency for retail businesses.

## Features
- User login authentication (Username: `admin`, Password: `123`)
- Dashboard with embedded IBM Cognos Analytics
- Overview of customer insights, sales analytics, and operational efficiency
- Team members and about us sections
- Dark mode toggle

## Getting Started
1. Open `index.html` in your web browser.
2. Log in using the credentials:
   - **Username:** admin
   - **Password:** 123
3. Explore the dashboard and its features.
